*  Description of the program
*******************************************************

This program implements a binary search tree and 
outputs the preorder, postorder, inorder, min etc.

*******************************************************
*  Source files
*******************************************************

Name: main.cpp
   Main program.  This is the driver program that calls
   functions to display the implementations of the 
   binary search tree such as postorder, preorder, etc. 

Name: BSTree.h
   Contains the definition for the class BSTree.  

Name: BSTree.cpp
   Includes functions to add node, print the binary
   search tree, find node with min and max key, find
   successor and predecessor to the node, and delete
   node.

Name: Node.h
   Contains the definition for the class DataType and the
   class Node.
