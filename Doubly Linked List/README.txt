*  Description of the program
*******************************************************

This program reads in a list of unsorted words into an 
implementation of a Doubly Liked List. It then 
searches the list from the beginning as well as the 
end and prints out how many searches were made to 
find the word. 

*******************************************************
*  Source files
*******************************************************

Name: main.cpp
   Main program.  This is the driver program that calls
   functions to search and sort the lists as well as 
   display the number of searches made. 

Name: DictEntry.h
   Contains the definition for the class DictEntry.  

Name: dictionary.cpp
   Contains definition for the class dictionary.

Name: guess.cpp
   Includes functions to read words from file, search
   words from the beginning of the list, search words
   from end of the list, print the list in reverse 
   alphabetical order, and displaying words.
