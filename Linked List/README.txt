*  Description of the program
*******************************************************

This program links a series of nodes together to 
implement a single linked list class. It implements
the concepts of a union and difference using a 
linked list.

*******************************************************
*  Source files
*******************************************************

Name: main.cpp
   Main program.  This is the driver program that calls
   functions to display the first and second lists 
   as well as the union and difference of the two 
   lists. 

Name: guess.h
   Contains the definition for the struct Node and 
   class linked list. It also contains functions for
   inserting/ deleteing nodes, overloaded operators
   and constructors/destrcutors.
